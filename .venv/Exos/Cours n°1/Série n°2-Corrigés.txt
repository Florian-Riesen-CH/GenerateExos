Exercice: 1
Qu'est-ce qu'un dictionnaire en Python ?

Un dictionnaire est un type de données modifiable en python qui stocke des paires de clés et de valeurs.


Exercice: 2
Quelle est la différence entre une liste et un tuple en Python ?

Le principal différence est que les listes sont modifiables alors que les tuples ne le sont pas.


Exercice: 3
En Python, qu'est-ce que le 'slicing' ?

Le 'slicing' est une technique pour extraire une partie d'une liste, d'un tuple ou d'une chaîne de caractères.


Exercice: 4
Expliquer le fonctionnement d'une boucle 'for' en Python.'

Une boucle 'for' est une boucle qui se répète un nombre prédéfini de fois. Elle est souvent utilisée pour parcourir les items d'une liste ou d'une autre structure de données.

Ecrire une fonction Python qui prend en paramètre deux listes de nombres et qui return une troisième liste contenant les éléments communs aux deux listes sans doublons.
