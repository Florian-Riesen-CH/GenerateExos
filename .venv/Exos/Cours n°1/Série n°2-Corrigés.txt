Exercice: 1
Qu'est-ce qu'une variable en Python ?

Une variable en Python est un espace de stockage qui permet de conserver une valeur temporaire pendant l'exécution du programme. On peut lui attribuer une valeur et ensuite l'utiliser dans les calculs ou les opérations.


Exercice: 2
Quelles sont les différences entre les conditions 'if' et 'elif' en Python ?

En Python, la condition 'if' est utilisée pour vérifier une seule condition. Si cette condition est vraie, alors le bloc de code correspondant est exécuté et les autres blocs de code après le 'if' sont ignorés. D'autre part, la condition 'elif' est utilisée pour vérifier une autre condition seulement si la condition précédente (if ou elif) est fausse. Si la condition 'elif' est vraie, le bloc de code correspondant est exécuté et les autres blocs de code après le 'elif' sont ignorés.


Exercice: 3
Quelle est la différence entre les opérateurs logiques 'and' et 'or' en Python ?

L'opérateur logique 'and' en Python retourne True si les deux conditions sont vraies. Sinon, il retourne False. L'opérateur logique 'or' en Python retourne True si au moins l'une des conditions est vraie. Sinon, il retourne False.

Exercice: 9
Écrivez une fonction Python qui calcule la factorielle d'un nombre donné. La factorielle d'un nombre entier positif n est le produit de tous les entiers positifs inférieurs ou égaux à n.

def factorielle(n):
    if n == 0:
        return 1
    else:
        return n * factorielle(n-1)

nombre = int(input('Entrez un nombre entier : '))
resultat = factorielle(nombre)
print('Le factorielle de', nombre, 'est', resultat)


Exercice: 10
Écrivez une fonction Python qui vérifie si une chaîne de caractères est un palindrome. Un palindrome est une chaîne qui se lit de la même manière de gauche à droite et de droite à gauche.

def est_palindrome(chaine):
    chaine_inverse = chaine[::-1]
    if chaine == chaine_inverse:
        return True
    else:
        return False

mot = input('Entrez une chaîne de caractères : ')
if est_palindrome(mot):
    print('La chaîne', mot, 'est un palindrome')
else:
    print('La chaîne', mot, 'n\'est pas un palindrome')


Exercice: 11
Écrivez une fonction Python qui trouve tous les nombres premiers inférieurs à un nombre donné.

def est_premier(nombre):
    if nombre <= 1:
        return False
    for i in range(2, nombre):
        if nombre % i == 0:
            return False
    return True

limite = int(input('Entrez un nombre entier : '))
for i in range(2, limite):
    if est_premier(i):
        print(i)
