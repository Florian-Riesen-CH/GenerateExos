Exercice: 1
Qu'est-ce qu'une variable en Python ?

Une variable en Python est un espace de mémoire dans lequel on peut stocker une valeur ou une référence à une valeur.


Exercice: 2
Quelles sont les types de variables disponibles en Python ?

En Python, il existe plusieurs types de variables, tels que les entiers (int), les nombres à virgule flottante (float), les chaînes de caractères (str), les booléens (bool) et les listes (list).


Exercice: 3
Qu'est-ce qu'une condition en Python ?

Une condition en Python permet d'exécuter un bloc de code uniquement si une certaine condition est vraie. On peut utiliser des instructions conditionnelles comme 'if', 'elif' et 'else' pour contrôler le flux d'exécution d'un programme.

Exercice: 11
Écrivez un programme Python qui calcule et affiche les N premiers termes de la suite de Fibonacci.

def fibonacci(n):
    fib_list = []
    a, b = 0, 1
    while len(fib_list) < n:
        fib_list.append(a)
        a, b = b, a + b
    return fib_list

N = int(input('Entrez le nombre de termes : '))
result = fibonacci(N)
print('Les', N, 'premiers termes de la suite de Fibonacci :', result)
