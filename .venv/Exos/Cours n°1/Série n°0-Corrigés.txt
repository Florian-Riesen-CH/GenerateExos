Exercice: 1
Qu'est-ce qu'une variable en programmation ?

Une variable est un espace de stockage qui contient une valeur ou une référence vers une valeur.


Exercice: 2
Quelle est la différence entre "if" et "elif" en Python ?

"if" est utilisé pour évaluer une condition et exécuter un bloc de code si la condition est vraie, tandis que "elif" est utilisé pour évaluer une condition supplémentaire si la condition précédente est fausse.


Exercice: 3
Quels sont les principaux types de données en Python ?

Les principaux types de données en Python sont: int (entier), float (nombre à virgule flottante), str (chaîne de caractères), bool (booléen), list (liste) et dict (dictionnaire).

Exercice: 9
Écrivez une fonction Python nommée "factorial" qui prend un nombre entier en paramètre et retourne sa factorielle. (Par exemple, factorial(5) doit retourner 120)

def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n-1)


Exercice: 10
Écrivez un programme Python qui demande à l'utilisateur de saisir une liste de nombres entiers et qui affiche à l'écran le plus grand nombre de la liste.

numbers = input('Entrez une liste de nombres entiers séparés par des espaces : ')
numbers_list = list(map(int, numbers.split()))
max_number = max(numbers_list)
print('Le plus grand nombre de la liste est :', max_number)
