Exercice: 1
Qu'est-ce qu'une variable en Python ?

Une variable en Python est un espace réservé en mémoire où nous pouvons stocker des valeurs. Elle peut être de différents types comme un entier, un flottant, une chaîne de caractères, un booléen, etc.


Exercice: 2
Quelles sont les différences entre un if et un elif en Python ?

Un if est une condition qui est évaluée et exécute un bloc de code si la condition est True. Si la condition est False, le bloc de code ne sera pas exécuté. Un elif est une autre condition qui est évaluée seulement si les conditions précédentes étaient False. Si la condition de l'elif est True, le bloc de code correspondant sera exécuté. Sinon, le bloc de code de l'elif ne sera pas exécuté.


Exercice: 3
Quelle est la différence entre une fonction intégrée (built-in) et une fonction définie par l'utilisateur en Python ?

Une fonction intégrée en Python est une fonction qui est déjà disponible dans le langage et peut être utilisée directement sans avoir besoin de la définir. Par exemple, la fonction print() est une fonction intégrée en Python. Une fonction définie par l'utilisateur est une fonction personnalisée créée par le programmeur selon ses besoins spécifiques. Elle doit être définie avant de pouvoir être utilisée.

Exercice: 11
Écrivez une fonction Python qui prend une liste en entrée et retourne une nouvelle liste contenant uniquement les éléments pairs de la liste d'origine.

def filtrer_elements_pairs(liste):
    elements_pairs = []
    for element in liste:
        if element % 2 == 0:
            elements_pairs.append(element)
    return elements_pairs
